# Alwan

Alwan is a Ruby Gem for colorizing text in the command line.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'alwan'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install alwan

## Usage

```ruby
puts 'foo'.colorize(:blue) + ' bar'
```
